#(3)时间分配的程序设计为：
将生成的网格化数据进行：月分配，周分配，时分配.可根据自己修改分配系数


#(4)程序使用说明：
运行程序之前，需要确定一下网格的一些信息（投影信息与网格信息），运用ArcGIS软件确定

（1）时间分配，需要修改生成文件的目标时间，与后一天的第一个小时
（2）物种分配，一般不需要修改
（3）生成二进制文件，需要修改时间与网格行列的信息
（4）生成NC文件，需要修改内容较多
